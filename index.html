HTML

Comandos importantes:

!DOCTYPE html: Declara que o documento é HTML5. html: A tag raiz do documento HTML. head: Contém informações sobre o documento, como o título e a ligação ao arquivo CSS. title: Define o título que aparece na aba do navegador. link rel="stylesheet" href="style.css": Liga o arquivo CSS ao HTML. body: Contém o conteúdo visível do site. h1: Define um título de nível 1 (o maior). p: Define um parágrafo de texto.

CSS: Estilizando seu Site

Comandos importantes

body: Define estilos para o corpo do site. h1: Define estilos para títulos de nível 1. h2: Define estilos para títulos de nível 2. p: Define estilos para parágrafos. font-family: Define a fonte do texto. font-size: Define o tamanho da fonte. color: Define a cor do texto.
